# Checklist+ Timer — Установка

## Что это
Кастомный Trello Power-Up с:
- вложенными чеклистами (задачи → подзадачи)
- таймером ▶/⏹ на каждый пункт и подпункт
- прогресс-баром по чеклисту
- суммарным временем на значке карточки

## Шаг 1 — Разместить файлы (бесплатно, GitHub Pages)

1. Создайте репозиторий на GitHub (например `trello-checklist-timer`)
2. Загрузите оба файла: `index.html` и `card-back.html`
3. Перейдите: Settings → Pages → Source: `main` / `root` → Save
4. Ваш URL будет: `https://ВАШ_ЛОГИН.github.io/trello-checklist-timer/`

## Шаг 2 — Получить Trello API Key

1. Откройте: https://trello.com/power-ups/admin
2. Нажмите **New** → придумайте имя (например «Checklist+ Timer»)
3. Укажите iframe URL: `https://ВАШ_ЛОГИН.github.io/trello-checklist-timer/`
4. Скопируйте **API Key**

## Шаг 3 — Вставить ключ в index.html

В файле `index.html` найдите строку:
```
appKey: 'YOUR_TRELLO_API_KEY',
```
Замените `YOUR_TRELLO_API_KEY` на ваш ключ и обновите файл на GitHub.

## Шаг 4 — Включить Power-Up в Trello

1. Откройте нужную доску
2. Меню доски → Power-Ups → Custom → введите URL: `https://ВАШ_ЛОГИН.github.io/trello-checklist-timer/`
3. Добавьте Power-Up

## Использование

- Откройте любую карточку → секция **Checklist+ Timer** появится внизу
- **＋ задача** — добавить пункт
- **＋ подзадача** — добавить вложенный пункт
- **▶** — запустить таймер, **⏹** — остановить, **↺** — сбросить
- Флажок ☑ — отметить выполненным
- На превью карточки видны прогресс (✓ 2/5) и суммарное время

## Альтернативный хостинг

- **Netlify**: перетащите папку на netlify.com/drop
- **Vercel**: `npx vercel --name trello-checklist-timer`
- Любой HTTPS-хостинг подойдёт
